| 附属属性 | 说明 |
|---|---|
| 治疗 | 增加自身生命值 |
| 加速 | 道具加速期间冷却时间读条增加100% |
| 冻结 | 道具冻结期间冷却时间暂停 |
| 充能 | 立刻增加一件道具的读条进度 |
| 剑势 | 剑势可用于发动蓄力攻击和振刀；蓄力攻击消耗剑势数量不等，振刀固定消耗50点剑势 |
| 振刀 | 剑势>=50点时，消耗50点剑势，让对方本次蓄力攻击不生效，且硬直3秒 |
| 减速 | 道具减速期间冷却时间读条减缓50% |
| 减伤 | 减少所有来源的伤害，不含中毒、灼伤伤害 |
| 暴击 | 有一定几率造成2倍伤害。 |
| 灼伤 | 每秒受到等同于当前灼伤层数的伤害，每秒层数减少10%。 |
| 中毒 | 每秒受到等同于当前中毒层数的伤害，无视护甲。 |
| 蓄力攻击 | 消耗剑势发动高额伤害攻击，可被振刀克制 |
