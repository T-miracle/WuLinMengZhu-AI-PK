# 游戏规则说明

## 1. 装备分类

游戏中的装备分为五大类：

| 分类 | 说明 |
|---|---|
| 近战武器 | 角色主要攻击装备，包含长剑、阔刀、长枪、匕首、双刀、太刀等武器类型 |
| 魂玉 | 提供战斗初始属性、触发效果或联动效果的 1*1 装备 |
| 药品 | 提供一次性或条件触发回复、增益效果的 1*1 装备 |
| 护符 | 也可称为道具，通常拥有发动间隔和主动/触发效果 |
| 护甲 | 提供护甲、防御、减伤或受击联动效果的防护装备 |

## 2. 背包规则

- 玩家背包为矩形格子，宽 4 格，高 5 格。
- 背包总格子数为 20 格。
- 装备放入背包时必须完整占用空格，不能超出背包边界，不能与其他装备重叠。
- 装备可以重复安装，只要背包空间允许，同名装备可以同时存在多件。
- 1 星、2 星、3 星装备在背包中不能存在相同星级的多件装备；例如两个 1 星装备不能共存，会自动合并成 2 星装备。不同星级的装备可以共存，例如 2 星和 1 星可以共存，3 星和 4 星也可以共存。
- 4 星装备可以在背包中同时存在多件。

## 3. 角色规则

- 默认角色生命值上限为 1000 点。

## 4. 装备占格

| 装备类型 | 默认占格 | 可旋转占格 |
|---|---:|---:|
| 护甲 | 2*3 | 3*2 |
| 长剑 | 1*3 | 3*1 |
| 阔刀 | 2*3 | 3*2 |
| 长枪 | 2*3 | 3*2 |
| 匕首 | 1*3 | 3*1 |
| 双刀 | 2*2 | 2*2 |
| 太刀 | 2*2 | 2*2 |
| 魂玉 | 1*1 | 1*1 |
| 护符 | 2*2 | 2*2 |
| 药品 | 1*1 | 1*1 |

## 5. 旋转规则

- 装备放置时可以旋转。
- 旋转会交换装备的宽高，例如长剑默认占用 1*3，旋转后占用 3*1。
- 正方形装备旋转后占格不变，例如魂玉、护符、药品、双刀、太刀。
- 旋转后的装备仍然需要满足背包边界和不重叠规则。

## 6. 装备属性

- 每件装备都有自己的属性信息。
- 属性可以包含基础数值、战斗初始效果、普通攻击效果、蓄力攻击效果、发动间隔、充能、冻结、治疗、护甲、减伤等内容。
- 装备属性以装备自身配置为准，同名装备也可以存在不同属性版本。
- 已有装备属性数据见《图片属性汇总表》。
- 附属属性含义见《附属属性说明表》。

## 7. 星级合成

装备星级可以通过同星级、同装备合成提升：

| 合成材料 | 合成结果 |
|---|---|
| 2 件 1 星装备 | 1 件 2 星装备 |
| 2 件 2 星装备 | 1 件 3 星装备 |
| 2 件 3 星装备 | 1 件 4 星装备 |

- 星级越高，装备属性越强。
- 目前仅 `冰魄`、`龙弧`、`烛神令` 提供独立星级档位，其余未分档装备默认按 `4` 星处理。
- 具体数值成长可由装备配置单独定义。
- 若没有单独配置成长表，默认应保持合成规则成立，但数值提升倍率需要后续补充。

## 8. 相邻属性规则

- 如果装备属性描述中包含“相邻”效果，只有与该装备在上、下、左、右方向直接接触的格子可以获得该效果。
- 对角线接触不算相邻，不能获得相邻属性。
- 对于占用多个格子的装备，相邻判定应以装备占用区域的外边缘为准。
- 只要目标装备任意一个占用格与效果来源装备任意一个占用格在上、下、左、右方向相邻，即可视为相邻。

## 9. 术语说明

| 术语 | 说明 |
|---|---|
| 治疗 | 增加自身生命值 |
| 加速 | 道具加速期间冷却时间读条增加 100% |
| 冻结 | 道具冻结期间冷却时间暂停 |
| 充能 | 立刻增加一件道具的读条进度 |
| 剑势 | 剑势可用于发动蓄力攻击和振刀；蓄力攻击消耗剑势数量不等，振刀固定消耗 50 点剑势 |
| 振刀 | 剑势 >= 50 点时，消耗 50 点剑势，让对方本次蓄力攻击不生效，且硬直 3 秒 |
| 减速 | 道具减速期间冷却时间读条减缓 50% |
| 减伤 | 减少所有来源的伤害，不含中毒、灼伤伤害 |
| 蓄力攻击 | 消耗剑势发动高额伤害攻击，可被振刀克制 |
